# 傻标（在线标注系统）

## 技术架构选型（初步）
### 前端技术架构选型
- 基础语言：React + TypeScript
- 状态管理：Mobx
- 路由管理：React-router-dom 4.x
- ajax 请求：axios + GraphQL
- 文件打包：webpack

### 后端技术架构选型
- 基础语言：Java
- 框架：SpringBoot
- 数据库：Mysql | MongoDB
- ORM: dbutil | hibernate | mybatis
- API 维护：Swagger

### 服务部署
- 服务器：阿里云
- 部署方式：k8s + docker
- 版本控制：github 私人项目
